# Dotfiles

## Installation

Install Packages
```sh
sudo pacman -S \
  hyprland \
  wofi \                   # App Launcher
  mako \                   # Notifications
  pipewire wireplumber \   # Screencapturing
```

Execute following command inside ~/.config
```sh
git clone git@github.com:Veridian211/.dotfiles.git .
```

## Switch Desktop Environment to Hyprland

Select Hyprland (Wayland) in the login screen.
